# go-memo-next

A simple `unrich` note app !!!

## Getting Started

### How to install

```bash
npm i -g pnpm
pnpm i
```

### How to start

```bash
pnpm dev
Open http://localhost:4000
```

### How to deploy

This project use gh-pages.  
Add environment `NEXT_PUBLIC_BE_URL` in `.env.production.local`

```bash
pnpm gh-deploy
```
